Dies ist ein einfacher Kalender, den ich mit Python und Kivy erstellt habe.
Benutzer können an beliebigem Datum Einträge erstellen, speichern, ändern und löschen.
Der heutige Tag, sowie Tage mit abgespeicherten Notizen werden farblich hervorgehoben.

Ursprünglich war dies ein Versuch, eine Anwendung für Android zu erstellen mit buildozer. 
Das hat leider bis jetzt trotz einiger Anstrengung nicht geklappt, deshalb
habe ich nun eine Version für Windows mit Pyinstaller erstellt. Wichtig ist, dass
die Datei save_file.json im gleichen Verzeichnis liegt wie die calendar.exe!
